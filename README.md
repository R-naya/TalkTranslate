# TalkTranslate

Projet personel de traduction vocale avec sous-titres.

L'objectif de ce projet est de développer une application permettant de traduire la parole d'un utilisateur et d'afficher le résultat sous forme de sous-titres. Pour le moment, le projet se concentre sur la mise en place du backend et la simulation de la transcription et de la traduction.

## Fonctionnalités actuelles

- Backend Python avec Flask
- API simple pour tester la communication client / serveur 
- Simulation de transcription
- Simulation de traduction
- Frontend HTML/JavaScript pour tester l'API
